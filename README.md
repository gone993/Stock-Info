# 📊 Stock Info

> 나만의 금융 정보 대시보드 - 실시간 환율, 지수, 공모주 정보를 한눈에!

## 🎯 프로젝트 소개

**Stock Info**는 개인 투자자를 위한 올인원 금융 정보 플랫폼입니다.  
원달러, 원엔 환율부터 코스피, 코스닥 지수까지, 투자 결정에 필요한 모든 정보를 세련된 인터페이스로 제공합니다.

"복잡한 금융 정보, 이제 우아하게 관리하세요." 💼

## ✨ 주요 기능

- 🌍 **실시간 환율 정보**
  - USD/KRW (미달러 원화 환율)
  - JPY/KRW (일본엔 원화 환율)
- 📈 **국내 지수 모니터링**
  - KOSPI (코스피 지수)
  - KOSDAQ (코스닥 지수)
